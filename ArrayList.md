#ArrayList
- Creating an ArrayList
    
```Java Creating ArrayLists
    ArrayList<Type> myArrayList = new ArrayList<Type>();```


- Add an object to the ArrayList

        myArrayList.add(object);

- return ArrayList length

      int length = myArrayList.size();

- return boolean if item in ArrayList

      boolean hasItem = myArrayList.contain(object);

- return index of object in ArrayList

      int objectIndex = myArrayList.indexOf(object);

- ArrayList resizes dynamically
- ArrayList holds objects, but will automatically wrap a primitive into an object
